# Requirements: MCP CLI Rust - v1.6 CLI Calling Conventions

**Defined:** 2026-02-14
**Core Value:** Reliable cross-platform MCP server interaction without dependencies.

## v1.6 Requirements: CLI Calling Conventions

Add bash-style calling conventions and fix JSON help text.

### Dynamic Flag Parsing

- [x] **ARGS-01**: Parse `--key value` as JSON field `{"key": "value"}` — ✅ SHIPPED v1.6
- [x] **ARGS-02**: Parse `--key=value` as JSON field `{"key": "value"}` — ✅ SHIPPED v1.6
- [x] **ARGS-03**: Parse JSON values directly: `--key {"a":1}` → `{"key": {"a": 1}}` — ✅ SHIPPED v1.6
- [x] **ARGS-04**: Fall back to JSON argument if no flags provided (backward compatibility) — ✅ SHIPPED v1.6

### Help Text Improvements

- [x] **HELP-01**: Fix error message to show valid JSON format `{"key": "value"}` — ✅ SHIPPED v1.6
- [x] **HELP-02**: Document both JSON and `--key value` formats in call help — ✅ SHIPPED v1.6
- [x] **HELP-03**: Show example for flag usage in call command help — ✅ SHIPPED v1.6
- [x] **HELP-04**: Update list command to show calling hint for tools — ✅ SHIPPED v1.6
